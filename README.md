# Portifolio_Analize_de_Dados_Python_AL
Nesse projeto imersivo foi uma preparação de analize profunda e identificando padrões no mercado de trabalho.
Utilizado Vscode, Google Colab, biblioteca Pancds para leitura e visualização dos dados com Python.
Foi feita de forma mais estruturada, preparando a base para análizes mais assertivas, focando identificar e lidar com dados faltantes, organizar colunas, aplicar filtros e gerar estatísticas descrtivas que ajudam a entender melhor os dados e extarair insights relevantes.
Criando gráficos para visualizar e explorar informações presentes nos dados em diferentes tipos de gráficos estatísticos como: histogramas, boxplots e gráficos de barras, para facilitar a interpretação e comunicar os resultados de forma mais clara.
